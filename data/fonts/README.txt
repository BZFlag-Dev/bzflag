The fonts in this directory were converted from true type fonts using
BZFlag's TextTool2.  TextTool2 is available in BZFlag's tools/ directory
of the source distribution.

Please contact one of the BZFlag developers if there are any questions
or concerns regarding existing fonts, the addition of new fonts, or
modifications.

Any fonts being added must be freely modifiable and freely
redistributable to be considered for inclusion.

Original TTFs may be found at the following locations:

Deja Vu fonts
http://dejavu.sourceforge.net/
